BasedPyright-Workflow 使用说明
==============================

配置文件位置：basedpyright-workflow/.bpr.json
当前 source_directory 设置为：autoBMAD/epic_automation

运行方式：
---------

1. 进入子目录：
   cd basedpyright-workflow

2. 运行命令：
   python -m basedpyright_workflow workflow

其他常用命令：
-------------

# 检查类型
python -m basedpyright_workflow check

# 生成报告
python -m basedpyright_workflow report

# 提取错误
python -m basedpyright_workflow fix

# 包含 Ruff 检查
python -m basedpyright_workflow workflow --include-ruff

# 指定源目录（覆盖配置）
python -m basedpyright_workflow check --path ./src

配置文件说明：
-------------
文件：basedpyright-workflow/.bpr.json
格式：JSON

{
  "project_name": "wpytQt_template",
  "source_directory": "autoBMAD/epic_automation",
  "ruff": {
    "enabled": true,
    "check_enabled": true,
    "format_enabled": true,
    "fix_enabled": true,
    "line_length": 88,
    "target_version": "py311",
    "select_rules": ["E", "W", "F", "I"],
    "ignore_rules": ["E501"]
  }
}

重要提示：
---------
- 所有命令都需要在 basedpyright-workflow 子目录中运行
- source_directory 是相对于项目根目录的路径
- 配置文件必须位于 basedpyright-workflow 子目录中
