## Dev Notes

### Technical Considerations
- Use `pyproject.toml` instead of `setup.py` for modern Python packaging
- Follow PEP 518, PEP 621, and PEP 660 standards
- Use `src/` layout to avoid import issues during development
- Ensure compatibility with common package managers (pip, poetry)

### Development Tools
- **Code Formatting:** black (line length: 88)
- **Linting:** ruff (extend recommended rule set)
- **Type Checking:** mypy with strict settings
- **Testing:** pytest with coverage plugin
