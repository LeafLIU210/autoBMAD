## Dev Notes
**Source:** Epic 1 - Core Algorithm Foundation

This story implements the core bubble sort algorithm. The implementation should be educational and clear, prioritizing understanding over optimization.

**Technical Requirements:**
- Standard bubble sort: repeatedly step through list, compare adjacent elements, and swap if in wrong order
- Pass through list until no swaps needed (list is sorted)
- Must be a pure function - not modify the original list
- Include detailed comments explaining each step
- Follow PEP 8 style guidelines

**Algorithm Logic:**
1. Start with first element, compare with next
2. If current > next, swap them
3. Continue to end of list (largest element "bubbles" to end)
4. Repeat until no swaps needed (list sorted)

**Testing Standards:**
- Test file location: tests/test_bubble_sort.py
- Use pytest framework with parameterized tests
- Test with: empty lists, single elements, already sorted, reverse sorted, random order
- Test with negative numbers, floats, and mixed numeric types
- Verify original list is not modified
- Achieve >95% code coverage
